Conventions to preserve

Reactivity

  • Raw data lives in ref(); derived data in computed() — never mutate a computed.
  • v-for keys are stable IDs (sku, order_number, month) — never the array index.
  • Validate dates before .getMonth() — some order_date values are nullable.

i18n & filters

  • Every visible string goes through t('namespace.key') from useI18n() — locale files in src/locales/{en,ja}.js.
  • Views consume the global useFilters() composable; never hold a private copy.
  • Backend filter params accept the literal 'all' sentinel — guard for it.
